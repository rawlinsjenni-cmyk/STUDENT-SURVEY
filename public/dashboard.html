How to Share the Survey

  1. Add a student using + Add Student
  2. Click the Copy Link button next to their name
  3. Paste that link into an email and send it to the student
  4. The student opens the link on any device (phone or computer)
  5. Their status changes to Completed once submitted
Each student gets their own unique link. Only you can see the responses.
